============
Installation
============


Stable release
--------------

To install cheesecake_kwalitee_index, run this command in your terminal:

.. code-block:: console

    $ pip install cheesecake_kwalitee_index

This is the preferred method to install cheesecake_kwalitee_index, as it will always install the most recent stable release. 

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for cheesecake_kwalitee_index can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/Michael-F-Bryan/cheesecake_kwalitee_index

Or download the `tarball`_:

.. code-block:: console

    $ curl  -OL https://github.com/Michael-F-Bryan/cheesecake_kwalitee_index/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/Michael-F-Bryan/cheesecake_kwalitee_index
.. _tarball: https://github.com/Michael-F-Bryan/cheesecake_kwalitee_index/tarball/master
